# Viagens - Landing Page

## Sobre o Projeto
Este projeto é uma landing page desenvolvida como parte de um desafio prático de desenvolvimento web. O objetivo principal é aplicar conceitos fundamentais de HTML semântico e CSS3, criando uma página interativa com o tema de viagens para vilarejos europeus, sem a necessidade de utilizar JavaScript.

## Funcionalidades e Requisitos Atendidos
* Estrutura construída inteiramente com tags semânticas do HTML5 (nav, main, section, article, footer).
* Banner principal contendo uma imagem de fundo remota e texto em destaque (overlay).
* Menu de navegação interno, onde os links direcionam o usuário suavemente para as respectivas seções na mesma página.
* Botões de "Voltar ao Topo" integrados nas seções para facilitar a usabilidade.
* Efeitos visuais e de transição (hover) aplicados nos links de navegação e botões, alterando a aparência original de forma fluida.

## Tecnologias Utilizadas
* HTML5
* CSS3

## Como Executar o Projeto
1. Faça o download dos arquivos ou clone este repositório para a sua máquina local.
2. Navegue até a pasta do projeto.
3. Dê um duplo clique no arquivo `index.html` para abri-lo no seu navegador padrão.

**Dica para desenvolvedores:** Caso utilize o Visual Studio Code, você pode instalar a extensão "Live Server". Basta clicar com o botão direito no arquivo `index.html` e selecionar "Open with Live Server" para ver as alterações no código sendo atualizadas em tempo real no navegador.
